# My Portfolio

A modern, interactive portfolio website built with React and Vite, featuring smooth animations and a sleek design to showcase projects and skills.

## ✨ Features

- **Interactive Animations** - Smooth transitions and motion effects using Framer Motion
- **Responsive Design** - Optimized for all device sizes
- **Modern Tech Stack** - Built with React 18 and Vite for fast performance
- **Custom Animations** - Engaging UI with scroll-based and spring animations
- **Fast Load Times** - Lightning-quick development and production builds

## 🛠️ Tech Stack

- **React** 18.2+ - UI library
- **Vite** 5.0+ - Next generation frontend tooling
- **Framer Motion** 11.0+ - Animation library
- **JavaScript** - ES modules

## 📋 Prerequisites

- Node.js 16+ 
- npm or yarn package manager

## 🚀 Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Sarthak-Mishra1104/my-portfolio.git
cd my-portfolio
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server with hot module replacement (HMR):

```bash
npm run dev
```

The portfolio will open at `http://localhost:5173` with live reloading enabled.

### Build

Create an optimized production build:

```bash
npm run build
```

The built files will be in the `dist/` directory, ready for deployment.

### Preview

Preview the production build locally:

```bash
npm run preview
```

## 🎨 Customization

The portfolio includes design tokens for easy customization:
- Colors (cyan, violet, pink, green, white)
- Animations and easing functions
- Typography and spacing

Modify these values in `src/App.jsx` to match your personal brand.

## 🌐 Deployment

You can deploy this portfolio to various platforms:

### Vercel (Recommended)
```bash
npm install -g vercel
vercel
```

### Netlify
Connect your GitHub repository to Netlify for automatic deployments.

### GitHub Pages
Build and push the `dist/` folder to your `gh-pages` branch.
